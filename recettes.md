# 📖 Catalogue de Recettes — BATCH Cooking Familial

> 20 recettes de base pour la rotation hebdomadaire.
> Respectent les règles métier : max 3 viandes/semaine, majorité végétal, simples, de saison.
> Grammages pour **1 adulte référence**. Multiplier par **3.6** pour la famille complète.

## 🥦 RECETTES VÉGÉTALES (12 recettes)

### R01 — Dahl de lentilles corail 🍂🌱
- **Type** : végétal | **Saison** : automne / hiver / printemps
- **Difficulté** : simple | **Prep** : 10 min | **Cuisson** : 25 min
- **Batch** : 100% lundi, se conserve 4 jours

| Ingrédient | Grammage adulte |
|-----------|----------------|
| Lentilles corail | 80g |
| Lait de coco | 100ml |
| Tomates concassées | 150g |
| Oignon | 60g |
| Ail | 5g |
| Cumin, curcuma, coriandre | 3g |
| Huile d'olive | 10ml |

**Étapes batch lundi :** Faire revenir oignon + ail, ajouter épices, lentilles, tomates et lait de coco. Cuire 25 min. Mixer partiellement. Refroidir + congeler en portions.
**Finition jour J :** Réchauffer 5 min, servir avec riz ou pain.
